# [Roman Numerals](https://codingdojo.org/kata/RomanNumerals/)

* Write a function that takes a number and converts it to the according roman representation.

For example, given the numbers, the function would return:
```
   1 ➔ I
   2 ➔ II
   3 ➔ III
   4 ➔ IV
   5 ➔ V
   9 ➔ IX
  21 ➔ XXI
  50 ➔ L
 100 ➔ C
 500 ➔ D
1000 ➔ M
```
